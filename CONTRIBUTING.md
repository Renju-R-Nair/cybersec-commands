# 🤝 Contributing

Thank you for helping improve this resource! Contributions are welcome from security researchers, students, and practitioners focused on **defensive security**.

---

## How to Contribute

1. **Fork** this repository
2. **Create a branch**: `git checkout -b add/new-tool-reference`
3. **Make your changes** following the style guide below
4. **Commit**: `git commit -m "Add: reference for <tool>"`
5. **Push**: `git push origin add/new-tool-reference`
6. **Open a Pull Request** with a clear description

---

## Style Guide

- Use clear, consistent Markdown formatting
- Include a brief description of the tool at the top
- Group commands logically with headers
- Add inline comments (`# explains what this does`) to all commands
- Include real-world defensive use cases where applicable
- Finish files with a `> ⚠️` disclaimer if needed

---

## What to Contribute

✅ New command references (e.g. `netcat.md`, `volatility.md`)  
✅ Corrections to existing commands  
✅ Defensive use case examples  
✅ Platform-specific variations (macOS, Windows)  
✅ Tips and tricks sections  

❌ Offensive-only content without defensive context  
❌ Tools that require unauthorized access  
❌ Proprietary or unlicensed content  

---

## Code of Conduct

Be respectful. This repo is for education and defense — contributions should reflect that spirit.
