# 🛡️ Cybersecurity Command Reference

A comprehensive reference guide for **defensive security** professionals and researchers. This repo covers essential commands, tools, and techniques used in network analysis, threat hunting, incident response, and more.

> Maintained for educational and defensive security research purposes.

## 🎯 Focus Areas

- **Defensive Security** — threat hunting, anomaly detection, hardening
- **Network Analysis** — traffic inspection, port scanning, flow analysis
- **Incident Response** — live forensics, connection auditing
- **System Hardening** — firewall rules, service auditing

All commands and tools listed in this repository are intended **strictly for authorized systems**, educational use, and defensive security research. Always obtain proper authorization before scanning or analyzing any network or system.
